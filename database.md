# Szczurux database

### PostgreSQL
```ini
DATABASE_URL="postgresql://USER:PASSWORD@HOST:PORT/DATABASE"
```

### MySQL
```ini
DATABASE_URL="mysql://USER:PASSWORD@HOST:PORT/DATABASE"
```

### MongoDB
```ini
DATABASE_URL="mongodb://USER:PASSWORD@HOST/DATABASE"
```

### SQLite
```ini
DATABASE_URL="file:./szczurux.db"
```
